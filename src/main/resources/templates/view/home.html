DEMO ABM

El siguiente demo posee dos ABM para conocer como es el intercambio de información entre las distintas capas.

El ABM País utiliza @PathVariable para recuperar los parámetros de la vista lo que permite utilizar el formato @GetMapping(“….. /{parametro}") Ej.: @PostMapping(“/usuario/baja/{id}”)al realizar una petición Get o Post Ej.: localhost:8080/usuario/baja/123456

El ABM Nacionalidad utiliza @RequestParam para recuperar los parámetros de la vista lo que permite utilizar el formato @GetMapping(“….. /") Ej.: @PostMapping(“/usuario/baja”) al enviar parámetros en una petición Get o Post Ej.:localhost:8080/usuario/baja?id=123456.

En las tablas que tiene las listas de País y Nacionalidad se puede observar la diferencias en las acciones para armar la URL y realizar las peticiones

En los formularios de alta, consulta y modificación de cada ABM se pueden observar dos formas de hacer las mismas acciones. Se deben observar los encabezados de los formularios, como se colocan los valores en los campos input y como se muestran los mensajes
